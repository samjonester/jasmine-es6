# jasmine-es6

Playground for ES6

## Prequisites
https://github.com/creationix/nvm
```
curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.30.2/install.sh | bash
nvm install stable
nvm alias default node
```

## Uses jasmine-es6 module
https://github.com/vinsonchuong/jasmine-es6

## Setup and Run
```
npm run setup
npm test
```
